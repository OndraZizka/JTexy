Class TexyModifier

Modifier processor.

Modifiers are texts like .(title)[class1 class2 #id]{color: red}>^
. starts with dot
(…) title or alt modifier
[…] classes or ID modifier
{…} inner style modifier
< > <> = horizontal align modifier
^ – _ vertical align modifier
